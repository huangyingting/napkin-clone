Slides UI Static Concepts

Four static interface directions generated from Slides-UI.md. Each keeps slide creation in the top Add menu, keeps the left rail focused on navigation and drag/drop reorder, places supplemental panels on the right, and uses a bottom dock for Slides, Notes, zoom, and view controls.

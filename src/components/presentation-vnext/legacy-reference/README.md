# Legacy presentation UI reference

This directory is a migration-only snapshot of `src/components/presentation`.
TypeScript and TSX files are copied with a `.legacy` suffix so they remain
searchable/editable reference material without joining the active vNext runtime,
typecheck, test, or import graph.

Use these files as source material when porting legacy editor shell, stage,
inspector, and interaction behavior to `DeckV7` components. Do not import from
this directory in product code.
